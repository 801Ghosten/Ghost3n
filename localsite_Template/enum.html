SNMP

    SNMP Enum

  • snmp-check 
    snmp-check [DIR_IP] -p [PORT] -c [COMM_STRING] 
     snmp-check 192.168.1.1 -p 161 -c public
  • SNMPwalk commands
    snmpbulkwalk -c public -v2c 10.10.11.136 . 
    
						snmpwalk -v [VERSION_SNMP] -c [COMM_STRING] [DIR_IP]
    
						snmpwalk -v [VERSION_SNMP] -c [COMM_STRING] [DIR_IP] 1.3.6.1.2.1.4.34.1.3 #Get IPv6, needed dec2hex
    
						snmpwalk -v [VERSION_SNMP] -c [COMM_STRING] [DIR_IP] NET-SNMP-EXTEND-MIB::nsExtendObjects #get extended
    
						snmpwalk -v [VERSION_SNMP] -c [COMM_STRING] [DIR_IP] .1 #Enum all 
    
						example: below outputs into human read able text to a file called snmpwalk10.164.119.104PT.txt
    
						snmpwalk  10.164.119.104 -c public -v1 . -OnT >> snmpwalk10.164.119.104PT.txt
  • Nmap 
    nmap --script "snmp* and not snmp-brute" targetIP
  • braa 
    braa @:.1.3.6.* #Bruteforce specific OID

    Exploiting

  1. run msf 
     search snmp 
     use 24 
     set RHOSTS targetIP 
     run

    verify if you are logged in

     search snmp 
     use 25 
     set RHOSTS targetIP 
     run

    verify read or right privledge

  3. run msf 
     search snmp 
     use 26 
     set RHOSTS targetIP 
     run

    verify if users were enumerated

  4. run msf 
     search snmp 
     use 27 
     set RHOSTS targetIP 
     run

    verify share files enumerated
